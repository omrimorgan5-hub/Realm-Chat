# Contributing

Thank you for considering contributing to Chat-Project.

How to contribute

- Fork the repository and create a feature branch (`git checkout -b feat/your-change`).
- Make small, focused commits with clear messages.
- Run existing tests before opening a pull request.

Coding guidelines

- Follow the existing Python style (PEP8).
- Keep functions small and single-responsibility.

Testing

- Run tests locally (set PYTHONPATH to `src`):

```powershell
$env:PYTHONPATH='src'; python -m pytest
```

Submitting a PR

- Open a pull request targeting `main` with a clear description and any steps to reproduce.

Maintainers will review and request changes if needed.
